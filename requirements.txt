# ESM-2 protein language model (evolutionary conservation scoring)
# Install ONE of the following:
#
#   Option A — Meta's official fair-esm package:
#     pip install fair-esm
#
#   Note: fair-esm may have compatibility issues with Python 3.14+.
#   If you get import errors, use Option B instead.
#
#   Option B — HuggingFace transformers (fallback):
#     pip install transformers torch
#
# Either package will be detected automatically by esm_bridge.py.
# The ESM-2 model (~30 MB) is downloaded on first use and cached locally.
#
# CamSol solubility scoring uses only the Python standard library —
# no additional packages required.

# ── CUDA / GPU acceleration ────────────────────────────────────────────────────
#
# For NVIDIA GPUs (ESM-2 inference acceleration):
#
#   RTX 5070 Ti (Blackwell, sm_120) — requires CUDA 13.0:
#     pip install torch==2.12.0+cu130 --index-url https://download.pytorch.org/whl/cu130
#   NOTE: cu130 DLLs may be blocked by Windows Application Control policies.
#   If blocked, PyTorch will gracefully fall back to CPU.
#
#   Ada Lovelace / Ampere (RTX 40xx, 30xx) — use cu126:
#     pip install torch==2.12.0+cu126 --index-url https://download.pytorch.org/whl/cu126
#
# ESM-2 bridge auto-probes CUDA with a test tensor op and falls back to CPU
# if hardware/driver mismatch is detected (see _probe_cuda_device in esm_bridge.py).
# CPU inference: ~27 ms/residue (fair-esm small model).
